# 矛盾分析法

> "捉住了这个主要矛盾，一切问题就迎刃而解了。"

## 核心要义

识别矛盾、抓住主要矛盾、区分矛盾性质。这是分析任何问题的底层框架。

## 适用场景

- 复杂问题分析
- 优先级决策
- 多因素权衡

## 分析步骤

### 1. 识别矛盾

列出问题中存在的所有矛盾点：
- 内部矛盾 vs 外部矛盾
- 主要矛盾 vs 次要矛盾
- 对抗性矛盾 vs 非对抗性矛盾

### 2. 抓主要矛盾

判断哪个是主要矛盾：
- 决定问题性质
- 影响其他矛盾
- 是解决问题的关键

### 3. 分析矛盾性质

区分矛盾的不同性质：
- 普遍性矛盾（普遍存在）
- 特殊性矛盾（具体问题具体分析）

### 4. 制定策略

根据矛盾性质制定解决方案：
- 主要矛盾 → 集中力量解决
- 次要矛盾 → 配合解决
- 矛盾转化 → 及时调整

## 使用提示

当面对复杂问题无从下手时，用矛盾分析法：
1. 先列出所有问题
2. 找出最关键的那个
3. 集中资源解决主要矛盾
4. 再解决次要矛盾
