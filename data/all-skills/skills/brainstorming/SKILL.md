# 头脑风暴 (Brainstorming)

## 概述

结构化的头脑风暴方法，帮助产生创意、解决问题、做出决策。

## 基础原则

```
┌─────────────────────────────────────────────────────────┐
│                  头脑风暴原则                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ✅ 数量优先: 想法越多越好，不要限制                       │
│  ✅ 延迟评判: 不要在产生想法时批评                        │
│  ✅ 异想天开: 疯狂的想法也可能有价值                      │
│  ✅ 组合创新: 把多个想法合并成新的                         │
│  ✅ 跟随思路: 一个想法可能启发另一个                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 流程步骤

### 1. 准备阶段
```markdown
## 头脑风暴准备

### 明确目标
- 要解决什么问题？
- 最终产出是什么？

### 确定参与者
- 谁来参与？
- 角色分工？

### 设定规则
- 时间限制？
- 发言顺序？
```

### 2. 热身阶段 (5 分钟)
```markdown
## 热身练习

### 快速发散
题目: "砖头能做什么？"

回答:
- 建造墙壁
- 敲门
- 压纸
- 纸张重量
- 锤子
- 装饰品
- 门挡
...
```

### 3. 正式风暴

#### 方法 1: 经典头脑风暴
```markdown
## 议题: 如何提高用户留存率？

### 想法收集
1. 游戏化机制（积分、徽章、排行榜）
2. 个性化推荐
3. 社交分享功能
4. Push 通知优化
5. 新手引导优化
6. 会员特权
7. 签到奖励
8. 内容运营
9. 社区互动
10. 离线功能

### 记录要点
- 不要评判，只记录
- 鼓励疯狂想法
- 鼓励组合创新
```

#### 方法 2: 逆向头脑风暴
```markdown
## 议题: 什么会让产品失败？

### 反向思考
- 用户会为什么抱怨？
- 什么会让用户离开？
- 竞品会怎么超越我们？

### 从失败中找方向
"如果问题是 X，那解决方案是..."
```

#### 方法 3: SCAMPER 法
```markdown
## SCAMPER 检查清单

S - Substitute (替代): 什么可以替代？
C - Combine (组合): 什么可以组合？
A - Adapt (改编): 怎么改编？
M - Modify (修改): 怎么修改？
P - Put to other uses (其他用途): 其他用途？
E - Eliminate (消除): 消除什么？
R - Reverse (反转): 反转会怎样？
```

### 4. 整理归纳
```markdown
## 想法整理

### 分类整理
| 类别 | 想法 | 可行性 | 优先级 |
|------|------|--------|--------|
| 功能 | 游戏化 | 高 | P0 |
| 功能 | 社交分享 | 高 | P0 |
| 运营 | 签到奖励 | 中 | P1 |
| 技术 | 智能推荐 | 中 | P1 |

### 筛选标准
- 用户价值
- 技术可行性
- 商业价值
- 开发成本
```

## 线上头脑风暴技巧

### 异步风暴
```markdown
## 异步风暴流程

Day 1: 发布议题 + 收集想法 (48h)
Day 3: 整理分类 + 投票
Day 4: 讨论 + 确定 Top 想法
Day 5: 行动规划
```

### 匿名收集
- 减少权威效应
- 让所有人都能发言
- 使用在线工具匿名提交

## 常见陷阱

### ❌ 避免
1. 领导先发言（压制了其他人）
2. 立即评判想法
3. 追求共识
4. 想法太少就结束
5. 偏离主题

### ✅ 推荐
1. 让基层先发言
2. 延迟评判
3. 鼓励少数派意见
4. 确保足够多的想法
5. 保持聚焦但允许发散
