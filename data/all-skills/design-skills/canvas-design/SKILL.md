# Canvas Design - 视觉内容创作

生成原创视觉设计，通过构图、色彩、图形传递主题意境。

## 设计理念

- **意境优先**: 设计服务于情感表达
- **文化融合**: 融合东方美学与现代设计
- **留白艺术**: 恰当的留白提升意境
- **原创性**: 拒绝模板化设计

## 支持类型

### 1. 海报设计
- 活动海报
- 产品海报
- 文化海报

### 2. 封面设计
- 书籍封面
- 专辑封面
- 文章封面

### 3. 社交媒体
- 公众号封面
- 小红书配图
- 微博配图

## 设计案例

### "草长莺飞"水墨海报

**主题**: 春天、生机、诗意

**设计方向**:
```
构图: 上下留白，中间主体
色彩: 水墨黑白 + 点缀嫩绿
元素: 
  - 飞鸟剪影（动态）
  - 柳枝垂下（柔美）
  - 远山淡影（层次）
  - 留白处题诗
```

**设计代码:**
```html
<div class="poster">
  <div class="background">
    <!-- 水墨渐变背景 -->
    <div class="ink-wash"></div>
  </div>
  
  <div class="elements">
    <!-- 远山 -->
    <svg class="mountain" viewBox="0 0 400 100">
      <path d="M0,100 Q100,20 200,60 T400,100" fill="rgba(0,0,0,0.1)"/>
    </svg>
    
    <!-- 柳枝 -->
    <div class="willow-branches"></div>
    
    <!-- 飞鸟 -->
    <div class="birds">
      <span class="bird bird-1"></span>
      <span class="bird bird-2"></span>
      <span class="bird bird-3"></span>
    </div>
  </div>
  
  <div class="poetry">
    <h1>草长莺飞</h1>
    <p>二月春风似剪刀</p>
  </div>
</div>
```

## 设计流程

1. **主题理解**: 解析关键词、情感基调
2. **风格定位**: 选择设计风格
3. **构图规划**: 确定视觉层次
4. **元素设计**: 图形、文字、色彩
5. **细节打磨**: 调整比例、间距
6. **输出交付**: 多格式导出

## 风格库

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 水墨风 | 留白、意境、淡雅 | 文化、艺术 |
| 极简风 | 少即是多、几何 | 科技、商务 |
| 复古风 | 怀旧、质感 | 文创、品牌 |
| 插画风 | 手绘、温暖 | 生活、教育 |

## 输出格式

- SVG 矢量图
- PNG 高清图
- PDF 印刷稿
- HTML 交互版
