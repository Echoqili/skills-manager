# First Customers

## Purpose

用逐个攻破的方式获取前100个种子用户。核心是**拒绝"等流量来了再说"的被动心态**，主动找到、联系、说服每一个潜在用户。

**适用场景：**
- 产品刚上线，没有用户
- 不知道如何开始推广
- 等待自然增长太久
- 需要第一批忠实用户建立口碑

## Key Concepts

### 冷启动策略

```
被动：等用户来 → 慢，死
主动：找用户去 → 快，活

主动获客矩阵：
          高质量用户    低质量用户
         ┌──────────┬──────────┐
主动联系  │  ★★★★★  │  ★★☆☆☆  │
         ├──────────┼──────────┤
被动等待  │  ★☆☆☆☆  │  ★★★☆☆  │
         └──────────┴──────────┘

策略：主动联系高质量用户
```

### 前100用户公式

```
目标 = 100个付费用户
时间 = 30-90天
方法 = 每天3-5个1v1沟通

计算：
100用户 ÷ 90天 = 1.1用户/天
实际：主动联系10个，约2个感兴趣，1个付费
需要：每天联系10-20个潜在用户
```

### 拒绝的心态

| ❌ 拒绝 | ✅ 拥抱 |
|--------|--------|
| 等SEO有流量 | 直接联系用户 |
| 等口碑传播 | 主动要求推荐 |
| 等产品完美 | 用现在的版本卖 |
| 等用户主动来 | 去找用户 |
| 不好意思打扰 | 帮助别人解决问题 |

## Application

### 1. 用户画像与获取渠道

```markdown
## 目标用户地图

### 核心用户（最容易付费）
- 特征：_____________
- 活跃地：_____________
- 痛点：_____________
- 现有方案：_____________

### 获取渠道优先级
1. [渠道1] - 为什么有效
2. [渠道2] - 为什么有效
3. [渠道3] - 为什么有效

### 接触话术模板
"Hi [名字]，
我注意到你[相关行为/发言]。
我做了一个[产品类型]来帮助[解决的问题]，
刚好能解决你提到的[具体痛点]。
有兴趣聊聊吗？"
```

### 2. 外展模板库

```markdown
## 邮件外展模板

### 模板1：问题共鸣型
Subject: [具体问题] - 3个月解决方案

Hi [名字],

看到你在[平台]问关于[问题]的问题，
我也有过同样的困扰。

最近我做了[产品]，专门解决这个问题。
[一句话价值主张]

想发你一个内测邀请，你方便吗？

[你的名字]

---

### 模板2：案例分享型
Subject: [客户成功故事] - 你可能感兴趣

Hi [名字],

[客户名]刚用[产品]解决了[问题]，
他们在[具体指标]上提升了[X]%。

感觉这个对你的[场景]也会有帮助，
想给你一个内测机会，可以吗？

[你的名字]

---

### 模板3：资源交换型
Subject: [你有X，我有Y]

Hi [名字],

我是[产品]的创始人。
我在研究[领域]，发现你的一些观点[具体引用]很受启发。

作为交换，我想给你提供[价值]，
你方便聊10分钟吗？

[你的名字]
```

### 3. 跟进序列

```
Day 0: 初次联系
Day 3: 跟进（没回复）
Day 7: 价值内容推送
Day 14: 再次跟进
Day 21: 最后尝试
Day 30: 移出序列（未响应）

记住：最好的销售是帮助，不是推销
```

## Examples

### Gumroad 前100用户

```
策略：Sahil Lavingia
1. 目标定位：Indie Hackers社区
2. 内容营销：分享创业故事
3. 1v1联系：在社区里DM每个新成员
4. 内测邀请：提供免费计划吸引早期用户
5. 口碑传播：用户满意后主动推荐

结果：3个月内获得100+付费用户
```

### B2B冷外展案例

```
产品：开发者工具
目标：前50个B2B用户

策略：
1. GitHub Top 1000仓库 → 找维护者
2. Twitter搜索相关话题 → 找活跃开发者
3. Dev.to文章作者 → 联系技术博主

话术：
"我看到你在[项目]用了[技术栈]，
我们刚做了个[工具]专门解决[问题]，
能给你节省[X]时间，要试试吗？"

结果：10%转化率，远超行业平均
```

## Common Pitfalls

1. **广撒网**：发给所有人，没有针对性
2. **纯推销**：只说产品，不理解用户
3. **不跟进**：发一封就没了下文
4. **太急躁**：想要太快成交
5. **忽视关系**：只想卖货，不想建立连接

## Success Metrics

```
前100用户里程碑：
□ Day 7: 获得第1个付费用户
□ Day 30: 获得10个付费用户
□ Day 60: 获得50个付费用户
□ Day 90: 获得100个付费用户

质量指标：
□ 续费率 > 80%
□ 用户主动推荐率 > 20%
□ NPS > 50
□ 客服工单 < 5%/月
```

## Daily Checklist

```
每天必须完成：
□ 联系20个潜在用户（邮件/DM）
□ 回复所有收到的消息
□ 记录每次沟通的反馈
□ 更新用户意向清单
□ 复盘今天的转化率

每周复盘：
□ 哪些渠道最有效？
□ 什么话术转化最好？
□ 用户最主要的3个问题？
□ 下周优先做什么？
```
