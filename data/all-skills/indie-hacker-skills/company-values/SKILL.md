# Company Values

## Purpose

用价值观筛选团队成员，确保团队高效协作。核心是**只招聘相信同样理念的人**，而非最聪明或最便宜的。

**适用场景：**
- 开始考虑组建团队
- 远程团队协作效率低
- 需要明确"我们是什么样的公司"
- 想要建立长期文化资产

## Key Concepts

### 10x 效率筛选器

```
只招比自己单打独斗效率10倍的人

原因：
- 1个10x开发者 > 3个普通开发者
- 10x设计师 > 5个普通设计师
- 节省管理成本

判断10x：
□ 完成同样工作，节省多少时间？
□ 质量比平均水平高多少？
□ 能独立解决问题，不需要监管？
□ 会主动改进流程？
```

### 远程异步文化

```
远程团队三原则：
1. 文档优先
   - 所有决策有文档
   - 异步沟通为主
   - 减少会议

2. 结果导向
   - 考核产出，不考核工时
   - 灵活工作时间和地点
   - 信任>监控

3. 主动共享
   - 每日更新进度
   - 遇到阻塞及时暴露
   - 分享学到的东西
```

### 价值观定义模板

```
## 公司价值观模板

### 核心价值（最多5个）
1. [价值1] - [具体行为定义]
2. [价值2] - [具体行为定义]
3. [价值3] - [具体行为定义]

### 行为准则
- 做正确的事，即使困难
- 主人翁心态，而非打工心态
- 透明沟通，直接反馈
- 持续学习，拥抱变化

### 招聘过滤器
"这个人是否相信我们的价值观？"

面试问题：
- 举一个你[价值观]的例子
- 如果[情况]，你会怎么做？
- 你为什么想加入我们？
```

    scores["skills"] = candidate.technical_score

## Examples

### Buffer 价值观案例

```
Buffer价值观：
1. 透明
   - 公开所有薪资
   - 公开公司财务
   - 默认公开会议

2. 自主
   - 无批准流程
   - 自己决定休假
   - 自己决定工作方式

3. 专注
   - 深度工作优先
   - 减少会议
   - 重视产出
```

### 招聘10x人才

```
招聘检查：
□ 这个人的工作质量是否10x平均水平？
□ 这个人能独立解决X问题吗？
□ 这个人能写出清晰文档吗？
□ 这个人会主动沟通问题吗？

问题示例：
"给我一个你主动改进流程的例子"
"描述一个你没有足够信息时怎么做"
"你为什么离开上一个工作"
```

## Common Pitfalls

1. **技能>价值观**：招了牛人，但破坏文化
2. **经验>潜力**：看简历，不看学习能力
3. **面试表现>实际能力**：被面试技巧蒙蔽
4. **速度>质量**：急着招人，降低标准
5. **同质化**：只招像自己的人，缺乏多样性

## Decision Framework

```
招聘决策检查：

□ 这个人相信我们的价值观？
□ 这个人的能力是否10x平均水平？
□ 我们能负担这个人的市场价？
□ 这个人能与现有团队合作？
□ 这个人能适应远程/异步？

全部Yes → 可以发offer
有No → 继续寻找
```
